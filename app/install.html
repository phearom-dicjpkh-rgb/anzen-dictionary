ដំឡើងកម្មវិធីវចនានុក្រម

ស្កេន QR ខាងក្រោម ដើម្បីបើកកម្មវិធី
https://phearom-dicjpkh-rgb.github.io/anzen-dictionary/
បើកកាមេរ៉ាទូរស័ព្ទ → ដាក់លើ QR → ចុចតំណដែលលេចឡើង

iPhone / iPad Safari

  1. បើកតំណក្នុង Safari
  2. ចុចរូប Share ⬆️ (ខាងក្រោមអេក្រង់)
  3. រំកិលចុះ → ជ្រើស “Add to Home Screen”
  4. ចុច Add → រូបតំណាង 辞 លេចនៅអេក្រង់ដើម

Android Chrome

  1. បើកតំណក្នុង Chrome
  2. ចុចប៊ូតុង ដំឡើង ដែលលេចនៅខាងលើកម្មវិធី
    (ឬចុច ⋮ → Install app / Add to Home screen)
  3. ចុច Install → រូបតំណាង 辞 លេចនៅអេក្រង់ដើម
បន្ទាប់ពីដំឡើងរួច កម្មវិធីបើកពេញអេក្រង់ដូច app ធម្មតា។
ត្រូវការគណនី (សិស្ស ឬ គ្រូ) ដើម្បីចូលប្រើ — សូមទាក់ទងអ្នកគ្រប់គ្រង។
